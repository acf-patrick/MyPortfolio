Mes Projets

Seed

Je voulais créer une code base facile à manier et refactoriser. Voici donc Seed un custom library écrit en C++, utilisant le modèle E.C.S pour Entity Component System, évitant ainsi le principe du GameObject. Je me suis en même temps inspiré de Godot sur ce projet.

Tiles Adventure

Tiles Adventure est un jeu de platforme écrit en C++. Le programme utilise SDL comme moteur de rendu, et la modification et concéption du niveau ont été fait sur Tiled, un logiciel gratuit facilitant le Level Design.

Fanorona

Une variante appelé Fanorontelo. Le programme utilise l'algorithme min-max pour évaluer le bon coup. Cette version peut être ainsi joué en solo.

NES Emulator

Comme son nom l'indique, ce projet est une tentative d'émuler la console NES. Pour le moment, on se focalise sur l'implémentation d'un debugger pour mieux réctifier les opcodes.

Chip8 Emulator

Bien que interpréteur est plus approprié que émulateur, ce programme lit un code source en CHIP-8 est applique le rendu via SDL.

Puissance 4

Puissance 4 est un jeu de stratégie abstrait. Les coups de l'ordinateur ne sont pas encore optimisés, mais le jeu peut très bien être joué à deux.

Morpion

Ceci est une implémentation écrit en Python, utilisant Pygame comme moteur de rendu. Le jeu peut également être joué en solitaire.

Flappy Bird

Flappy Bird est un jeu pour mobile, initialement popularisé sur Android. Cette implémentation utilise SFML comme moteur de rendu.

Numerical Analysis

Étant passioné par les mathématiques, j'ai écrit ce petit library en guise de pratique dans mon étude sur l'analyse numérique. Ce projet propose des fonctions qui approximent les zéros une fonction, calcul l'approximation polynomiale (Lagrange, Newton), intégration et différenciation numérique.
